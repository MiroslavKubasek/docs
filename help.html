Where To Get Help

If you are stuck or have a question about jmpress.js, there are a few places to get help:

  • Github Issues Create a new issue. We take support questions there as well. Please provide a link to an example of your issue, if possible.

  • Google Group Ask/Answer questions regarding jmpress.js.

  • #jmpress on irc.freenode.net Ask/Answer questions via IRC.

  • Google+ +1 us on Google+ and get updates about the project.

  • @shamakry Tweet your question or notification directly.
